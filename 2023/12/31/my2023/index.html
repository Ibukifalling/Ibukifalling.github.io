2023年度摆烂记录

前言

不知不觉把年度总结拖了这么久（笑）

我想，如果要我为我的2023选出一个年度关键词，那么“摆烂”是最能概括我这一年的词语了😆

自从接受了自己的平庸和愚钝之后，似乎一切东西都不那么重要了——美好的事情到底发不发生在我身上也无所谓了，我就是一个很随遇而安的人，没有什么远大的目标和追求，只想过的开心一点

然后，和往常一样，我要放在这里一首歌作为BGM

（这首歌的歌词很适合我的心情）

My OKR

以下内容由chatgpt生成：

OKR（Objectives and Key Results）是一种目标管理框架，旨在帮助组织和个人明确目标并衡量其实现的程度。这个框架通常用于提高团队和组织的效率，确保所有人都朝着共同的目标努力。

OKR由Intel公司的前首席执行官安迪·格鲁夫（Andy Grove）在20世纪70年代末首次引入，并在Google等科技公司得到广泛应用。该框架的核心概念分为两个主要部分：

Objectives（目标）： 这是描述期望实现的具体目标的句子或短语。Objectives通常是具有挑战性和激励性的，它们应该激发团队和个人的积极性。

Key Results（关键结果）： 这是衡量实现目标进度的具体、可量化的指标。Key Results应该清晰、可度量，并能够反映目标的完成程度。

我最早看到OKR这个词应该是在飞书里，这是一种类似KPI的东西。

不过比起KPI这种旧时代的的产物，OKR显然更具有现代感。并且在某个瞬间我突然发现，给人生划分一些Objectives显得那么自然且合理——活着本来就是用各种各样的手段去达成各种各样的目的，与开公司似乎也没什么不同。

所以今年的年度总结我准备用OKR的方式来写

Objective-活着

活着当然是人生一大要事。当然，这里的活着并不仅仅是指饿不死，更多是指在社会意义上作为一个“人”而活着。再说的直接一些，就是去做“我该做的事”

尽管我常常会对到底什么是“该做的事”产生疑问，不过大部分时候我还是会想像个正常人一样去体验人生的

Key Result-人际

嗯，人际交往当然是活着的必需品啦。虽然完全没有社交也不会死人，但我当我想起七月上班时整天没有社交的样子，特别是晚上倒在床上想喘口气时，发现嘴巴已经因为长时间紧闭而粘在了一起，还是会希望有人可以和我说说话。

话虽如此，不过我的人际交往能力还停留在宝宝水平😆

23年我的社交能力并没有什么进步，依然贯彻着相对较低的情商以及一些莫名其妙的傻逼言论和行为（甚至有病情加重的倾向）

我严重怀疑病情加重是因为我的主要网络社交群体从🍺群变成了💩群

粪坑群的起源说来话长，长话短说就是由于有一群人经常在学校Steam群发表抽象低素质言论导致群主出手管控。但是人总得拉屎，为了不让大街上变得💩满为患，一定要有一个厕所——于是吴豪滨自立为粪坑王，组建了💩群。

💩群的群规只有两条：不许鉴证，不许恶意攻击，除此之外自由开火。底层逻辑决定上层表现形式，💩群的建立初衷与管理理念注定了💩群的聊天内容充斥着大量无意义的对话，包括但不限于（以下词语均为褒义）玩抽象/人身攻击/诋毁/造谣/恶心人/地域黑/群嘲/霸凌/权限/玩烂梗等。

💩群建立之初的群名为5team唐人街，随着烂梗数量越来越多，每多一个新的烂梗，就会在原有的群名中增加一个词语，截止本文写成时，💩群的完整群名是《四爱死舔狗聊👩🏻刷📱🦕🦖ckw偷拍通宵孤立权限report智商低唐人街》

至于现实里的社交，则属于是一直想迈出而一直迈不出。

我自认为并不是一个很社恐的人，比起害怕和陌生人交谈，我如此内向的原因可能更多是虚无主义的侵扰——所有相遇都为了分离，为什么还要开口说出第一句“你好”？

尽管知道做人不能太虚无主义，但是对世间一切事物感到悲观的底色早已深深植在我的性格之中，比起逼自己改变，我更乐意选择坦然接受——不去刻意寻求，随遇而安就好（很符合摆烂精神）😎

Key Result-学业

嗯，其实我觉得这个可以直接跳过的，我的绩点长什么样，大家应该都很清楚。。。。。。😢

想读研的想法确实也只停留在了想想，条条大路通读研，但似乎没有任何一条适合我

不出意外的话，我的学业生涯应该是快走到头了

Key Result-工作

既然学业生涯走到头了，那么找个工作就是对于活着来说比较重要的事情了

七八月份的时候，通过学长的内推去了阿里云实习

和我同期投的nama没有过，我们百思不得其解，因为他的简历跟我可以说是一模一样。。。各种比赛和项目基本上都是重合的，而且nama更擅长Java，按理来说更适配一些。
我们思考了很久之后得出结论：是因为nama绩点太高了，HR怕他保研跑了。。。

阿里云的工作最爽的是有巨量的内部文档可以看，之前在字节实习时的Web洞都比较常规，文档什么的也和网上可以找到的那些教学没有太大差别，阿里云的文档则有很多涉及到云安全的漏洞细节可以学习

小插曲：
入职之前有要求上传一张工牌照片，页面上特别标注了要拍的快乐一点。在我的想象里，工牌应该是一种类似于学生证，平时不需要拿出来，拿出来也只是给保安看的东西。所以本着“你要我快乐，那我就快乐给你看”的想法，发动所有面部肌肉拍了一张很快乐的照片。

然后发现这个照片居然是员工帐号的头像

因为这个照片完全不像本人，我甚至刷脸过不了门禁（大无语）

在阿里云的日子没有挖出什么比较重要的洞，顶多有一些边缘站的rce。不过通过内部文档和耳濡目染的方式学到了很多东西，包括我的师兄也非常热情，会常常和我进行一些概念上的讲解和架构方面的安全分析，受益良多

之后九月份的时候，因为北邮的傻逼课程安排不得不回学校

然后一方面因为实习期间的业绩确实不是很出彩，另一方面阿里云本身可能也不太好进，感觉转正没有抱多大希望，就去投身秋招了。

秋招的话，笔试挂了很多。。。（我有考试恐惧症）

过了笔试的奇安信和深信服前几面技术面聊的都挺开心的，hr面给我挂了。

当我正准备喜提秋招0offer备战春招的时候，阿里云发邮件告诉我转正过了（喜提🥬），找工作的事终于可以告一段落了。

Key Result-自嘲

“还差一点 就能跟他们一起取笑我自己” –《老张》

然而我并不是老张😆我可以随便取笑我自己

这一年实在是做了太多值得嘲笑的事，包括但不限于：

  • 出于学术研究目的（真是学术研究）下载了某女性专用交友软件，结果刷到认识的人还留了访客记录，被对方质问“你是出于什么目的”

  • 被多次拉黑，被拉黑的理由总结一下就是“我是傻逼”



  • 多次骚扰有妇之夫无果而终

可能确实有点表演型人格吧，不过表演已经成为我生命中不可或缺的快乐了。百度说表演型人格一般25岁以下比较常见，看来我还有3年时间可以快乐😆

Objective-以后也要活得开心

想要一直活的开心并不是一件很容易的事情，毕竟未来总是未知

Key Result-学🎸

学🎸的念头其实来源于我某一天上网冲浪时看到这么一句话

音乐形成了一个至关重要，在intp生命中扮演着唤醒不发达的Fe影子的关键角色。毋庸置疑，正是Fe在音乐的体验中给intp带来了生命的火花引入到真正快乐的感觉。

“唤醒不发达的Fe影子”实在是很符合我的需求，所以不得不相信一下赛博星座了

经典吉他笑话一则

目前我的练琴还是处于一个三天打鱼两天晒网的状态，希望能在我的未来生活里占据一些重量吧

Key Result-爱情

很喜欢jbr的一句话：

爱情也算未来人生的一个比较重要的部分吧，可是亲密关系这种东西实在是太玄乎了，我小小的脑袋尚且难以理解，也并没有真正投身过一段亲密关系

至少目前为止，组成我的心理内核的还是理性与冷漠，这对于发展亲密关系来说并不好，但即使我有意想要改变，这也是一个漫长的过程

所以说到底还是继续随缘吧

Key Result-思考未来

我还是觉得人们需要经常停下来想一想自己到底是什么样的人/自己到底想要什么。

我没有什么很远大的理想和抱负，也没有一定要按照某种特定的方式去度过人生的执着，我的未来依旧开放而广阔（我不喜欢用迷茫这个词，在我看来人生是不迷茫的，因为人生的终点一定是死亡，只是过程不一样而已）

如果让现在的我来说，我会说dreamlife就是可以每天都躺在沙发上看电影。但未来的我一定会有另外的想法，到那时我会想做什么、会想要什么，现在的我不得而知。

不过现在的我知道，未来的我一定不会后悔我曾经做过的事（我上一次对自己的所作所为后悔要追溯到小学了），这可能也是我能如此自由任性的基础

所以简单一点，我只要不断前进就好了，不用顾忌那么多

后话

因为最近比较忙（后面应该也会挺忙的）导致年度总结拖了很久

写到一半的时候会想，年度总结到底该写什么呢？什么要写什么不写？想来想去，还是决定延续我一开始的想法——“记录我生命中各种各样的瞬间，让我以后看到的时候能会想起当初”

我也不知道许多年以后我还会不会回头来看自己写的年终总结，但对于现在的我来说，只要把我现在的心情留下来就够了。
